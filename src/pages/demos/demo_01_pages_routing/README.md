# Demo 01: Pages & Routing

**Concepts:** File-based routing, static/dynamic routes, params, catch-all routes

## Files
- `index.astro`: Home page
- `about.astro`: Static route
- `blog/[slug].astro`: Dynamic route
- `catchall/[...all].astro`: Catch-all route

## How to Show
- Add/remove `.astro` files and show instant route changes
- Visit `/blog/astro`, `/catchall/anything/here` to demo params
